# Bit Debugger

Bit Debugger é um jogo de plataforma educacional que apresenta conceitos
iniciais de programação de forma interativa e acessível para crianças.

O projeto foi criado para a **10ª MostraTech**, realizada em junho de 2024
pela [FPF-Etech](https://www.fpf-etech.com/). O jogo foi desenvolvido em
Python com Pygame como parte do processo de aprendizado da linguagem.

## Jogar

O jogo pode ser executado diretamente no navegador:

### [Jogar Bit Debugger](https://antonio-pss.github.io/bit-debugger/)

## Como jogar

- Use `A` e `D` ou as setas direcionais para se movimentar.
- Use `Espaço` ou a seta para cima para pular.
- Pressione `E` para interagir com dicas.
- Pule sobre os inimigos para derrotá-los.
- Responda aos desafios para avançar.
- Pressione `Esc` para abrir ou fechar o menu.

Os cafés representam as vidas do personagem.

## Tecnologias

- Python
- Pygame CE
- PyTMX
- Pygbag e WebAssembly para execução no navegador
- Tiled para construção do mapa

## Executar localmente

É necessário utilizar o Python 3.12.

Instale as dependências:

```bash
python -m pip install -r requirements.txt
```

Execute o jogo:

```bash
python main.py
```
